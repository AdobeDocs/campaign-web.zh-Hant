>id="acw_explorer"
>title="探索工具"
>abstract="TBC"

Web UI中瀏覽器的菜單，以瀏覽控制台內容：控制台導覽樹第二個檢視，以及包含篩選器的左側功能表清單。 瀏覽器會從主控台提供真正的資料夾階層。 請務必在子資料夾中找到您的傳送。 所有清單都可以唯讀存取。 無建立/編輯。 您可以設定清單（列）。 所有架構欄位、連結的表均可用。
