# AI支援的內容說明 {#using-ai}

透過產品內容說明方塊中提供的問答AI支援的內容說明，根據產品檔案和您目前的產品內容，取得快速的AI產生的答案。

>[!CAUTION]
>
>此功能可在 Alpha 版本中使用，如有變更，恕不另行通知。
>
>此功能僅以&#x200B;**英文**&#x200B;提供。此版本不支援其他語言。 因此，請確定 [您偏好的語言](connect-to-campaign.md#language-pref) 設為英文後再使用此功能。


此 **Gen AI的情境式說明** 內嵌在內容說明中的內容，革新了檔案搜尋和回答操作方法問題，讓您能夠輕鬆地在龐大的檔案存放庫中篩選，立即找到您需要的精確資訊。

有了Campaign Gen AI的功能，此元件可改變您的體驗，讓資訊擷取和問題解決變得輕而易舉。 無論您是在複雜的工作中尋找指引，還是瀏覽廣泛的檔案，AI支援的情境式說明都是您的最終夥伴，提供您無與倫比的效率與準確性，每次互動中都一樣。

## 答案 {#answers-ai}

Adobe Campaign AI支援的情境式說明可提供協助，但您應驗證事實。 它可能會輸出不正確的資訊、不準確或誤導性的答案。請務必檢查每個說明方塊中連結的產品文件。

人工智慧和機器學習模型會隨著時間而不斷改進，以更好地解決特定的使用案例。Campaign AI支援的情境式說明在未來幾個月中會持續改善。 我們強烈建議您在閱讀答案時隨時使用 ?? ?? 按鈕，以向我們的工程師標記任何不當的回應。

## 建議  {#recommendations-ai}

在AI支援的內容說明中提出問題時，會考慮目前的產品內容（即您所在的畫面）以及目前說明方塊的內容。

因此，最佳做法是提出與使用者介面內容相關的問題。例如，如果您需要了解如何在工作流程中傳送推播傳遞，請瀏覽至「**工作流程**」左側選單項目，並在該特定內容中使用內容說明方塊。若要深入了解報告量度，請瀏覽至行銷活動或傳遞報告，然後從該處開始。


此外，若要充分利用AI支援的情境式說明，我們建議您在提出問題時：

* 盡可能精確和具體，避免歧義用語。如此，答案將會更準確、更有幫助。
* 為協助AI支援的內容說明找到正確的資訊，請儘量新增您想要瞭解的詳細資訊。
* 重新措辭並改善您的問題，以便獲得更好的答案。如果答案不準確或沒有用，請嘗試不同的方法並加入相關內容以獲得更好的結果。
* 使用適當的術語和措辭，採用中性語氣。
* 對答案進行評分並提供回饋，以便我們的工程師從您的經驗中學習並改進結果。
